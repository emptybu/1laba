**Цель работы**

Освоить основы контейнеризации приложений с использованием Docker и Docker Compose.

Выполненные задачи

**Основные этапы:**

Создание Dockerfile для Python приложения
Разработка Flask приложения с подключением к PostgreSQL
Настройка docker-compose.yml для оркестрации контейнеров
Запуск многоконтейнерного приложения (Flask + PostgreSQL)
Настройка сетевого взаимодействия между контейнерами
Техническая реализация:

Flask приложение - веб-сервер на Python с маршрутом /
PostgreSQL - база данных в отдельном контейнере
Docker Compose - управление многоконтейнерной средой
Сетевое взаимодействие - автоматическое подключение между сервисами
Результат

**Успешно развернуто веб-приложение** в Docker контейнерах с подключением к базе данных PostgreSQL. Приложение доступно по адресу http://localhost:5001 и отображает статус подключения к БД.
